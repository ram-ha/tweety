## 📌 프로젝트 실행 방법

1. Clone the repo

```javascript
$ git clone https://github.com/ram-ha/tweety.git
```

2. Install NPM packages

```javascript
$ npm install
```

3. Getting Started

```javascript
$ npm run dev
```
